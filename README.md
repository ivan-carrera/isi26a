# ISID223 Introducción a los Sistemas de Información
### Repositorio de Ejercicios y Talleres Prácticos
**Escuela Politécnica Nacional** | Facultad de Ingeniería de Sistemas

---

Bienvenido al repositorio oficial de las actividades prácticas, laboratorios y talleres de la asignatura **Introducción a los Sistemas de Información**. El objetivo de este espacio es consolidar los conceptos teóricos fundamentales de la infraestructura tecnológica, la gestión de datos y el rol estratégico de los SI en las organizaciones a través de la experimentación directa y la resolución de problemas reales.

## Objetivos de Aprendizaje

* **Aplicar el enfoque de sistemas:** Analizar y estructurar problemáticas organizacionales vinculadas a las Tecnologías de la Información (TI).
* **Fundamentación tecnológica y de software:** Traducir conceptos de datos e infraestructura mediante el uso de herramientas como Python y manipulación de datasets.
* **Resolución de Casos de Estudio:** Diseñar soluciones y arquitecturas de SI para diversas situaciones empresariales complejas.
* **Colaboración y Desarrollo:** Fomentar el trabajo en equipo multidisciplinario a través del control de versiones (Git) para la gestión del proyecto de la asignatura.

---

## Tecnologías y Prerrequisitos

Para ejecutar los laboratorios de código y talleres de este repositorio, asegúrate de contar con el siguiente entorno configurado:

1. **Git:** Para clonar el repositorio y gestionar las ramas de tu equipo de proyecto.
2. **Python 3.8+:** Entorno de ejecución de los laboratorios de datos.
3. **Librerías de Python requeridas:**
   ```bash
   pip install pandas matplotlib jupyter
   ```

4. Herramientas de Diseño (Recomendadas para el proyecto): Figma, Balsamiq o Draw.io para mockups y diagramado de bases de datos.

## Bibliografía:

Toda la fundamentación teórica que respalda estos ejercicios prácticos está estrictamente alineada con la literatura oficial de la asignatura:

* Stair, R., & Reynolds, G. Principles of Information Systems.
* Rainer, R. K., Prince, B., & Cegielski, C. Introduction to Information Systems.

Se recomienda revisar los capítulos asignados en la planificación de la asignatura antes de iniciar cada laboratorio práctico.
